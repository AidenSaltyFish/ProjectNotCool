Declaration

This is a course project in fulfillment of requirement of course 02806, Social data analysis and visualization of Semester Spring 2022, in Technical University of Denmark.

This website is developed via Hugo, with template from Robert Austin and edited a little with love by Kewei Du.

ProjectNotCool is fully open-source. Please visit the Github page for more information.
